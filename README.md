## 基礎（basic/）
- 組み合わせの漸化式（combi/）
$_{n}C_{r}$を以下の漸化式で求める.
```math
\begin{cases}
    {{_{n}C_{r}} = \frac{n - r + 1}{r} {_{n}C_{r-1}}} \\
    {{_{n}C_{0}} = 1}
\end{cases}
```
